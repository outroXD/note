# テストの観点覚書
## API
* 正常系ケースとして、必須パラメータのみ入力した場合/必須パラメータとその他パラメータ全てを入力した場合。
* 日付のフォーマットが意図したものと違う場合の動作の確認。
* 日付が一致/異なる場合の動作の確認。
* Content-Lengthの長さを想定より短く設定した場合。
